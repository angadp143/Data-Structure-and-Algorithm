## Data Types and Conversion

### Data Types
```cpp
#include <iostream>
using namespace std;
int main()
{
    int age = 21;
    cout << age << endl;
    return 0;
}
```

### TypeCasting
```cpp
#include <iostream>
using namespace std;
int main()
{
    double price = 100.9292;
    int newPrice = (int)price;
    cout << newPrice;
    return 0;
}
```
